# Novedades DinaZen — Enero 2026

Versión **10.11.0.6** — arranque público de la librería de componentes Blazor.

### Qué incluye

- **KPIs y estadísticas** — `KpiCard`, `StatsDisplay`, `StatItem`.
- **Formato de datos** — componentes `Span*` para fechas, bytes, dinero, duraciones.
- **Avatares, badges y loaders** — lo básico para cualquier panel.
- **Rangos de fecha y selectores** — `DateRangeCard`, `DateRangeSelector`.
- **Gantt** genérico con su toolbar.
- **Visor de Office** y **resaltado de código** con `highlight.js`.
- **Wizard por pasos** (`Stepper`).
- **App demo** con Docker y README público con la API JS.
